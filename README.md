## Adoptation

This template was wriiten with an idea kept in mind, that it will be used along with `Django` or `Laravel`

Includes:

- `lodash`
- `redux` with `redux-thunk`

Modules:

- Modal
- Localization

Predefined:

- `.eslintrc`
- `.editorconfig`
- `.gitignore`
- `tsconfig.json`

- `Fuse` - For searching values
- `recoil` - For compact contexts
